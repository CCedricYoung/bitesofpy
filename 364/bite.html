Lorem Ipsum is text used to demonstrate layout in documents (such as web pages) without distracting reviewers with real content.

This article describes it nicely. Innovative developers have made a number of different lorem ipsum generators to mix it up a little, using themes like bacon or pirates.

In this Bite, we want to create lorem ipsum text using words we scrape using BeautifulSoup from a Wikipedia featured article of the day.

We don't use the live page, we work with this static copy (which will be loaded in) so we can reliably test your code. While not part of the Bite, a bonus next step would be to allow a date argument and use this to choose the article from which to grab those words.

HINT: BeautifulSoup takes in a parameter to tell it what type of parser to use. I suggest using html.parser.

Make sure the sentences are well-formed - with an uppercase letter at the start of each sentence and a period at the end of each.
